# UI/UX設計 (UI_UX_DESIGN)

## 1. 画面遷移図
- **目的:** アプリケーションの主要な画面と、ユーザーの操作による画面間の関係性を示す。
- **図:**
  ```
                                        +-------------------------+
                                        | (Chrome拡張)            |
                                        | WebクリッパーUI         |
                                        +-------------------------+
                                                  | (カード作成)
                                                  v
  [トップページ / ダッシュボード] <---- (完了) -- [カード登録/編集画面]
    |
    | (今日の復習を開始)
    v
  [復習画面] ----------------------------> (セッション終了) --> [トップページ / ダッシュボード]
    |  ^
    |  | (次のカードへ)
    +--+
  ```
- **主要フロー:**
  1.  **学習フロー:** `トップページ` → `復習画面` → `トップページ`
  2.  **登録フロー:** `トップページ` → `カード登録/編集画面` → `トップページ`
  3.  **外部登録フロー:** `WebクリッパーUI` → `カード登録/編集画面`

## 2. ワイヤーフレーム (Wireframes)
- **目的:** 主要な各画面のレイアウトと、配置されるUIコンポーネントを定義する。

### 2.1. 【画面1】トップページ / ダッシュボード
- **役割:** 学習へのエントリーポイントであり、モチベーションを維持するための情報を集約する。
- **レイアウト:**
  ```
  +----------------------------------------------------------------------+
  | Header: [Memory App]                               [設定 ⚙️]         |
  +----------------------------------------------------------------------+
  |                                                                      |
  |  +--------------------------------+  +-----------------------------+ |
  |  | <今日の復習>                   |  | <学習サマリー>                | |
  |  |  - `今日のカード`: 25枚         |  |  - `総カード数`:  150枚      | |
  |  |                                |  |  - `連続学習日数`: 12日 🔥    | |
  |  |  [ 学習を開始する ▶️ ] (Button) |  |                             | |
  |  +--------------------------------+  +-----------------------------+ |
  |                                                                      |
  |  --- デッキ一覧 ---------------------------------------------------- |
  |                                                                      |
  |  [ + 新規カードを登録 ] (Button)  [ + 新規デッキを作成 ] (Button)     |
  |                                                                      |
  |  - [デッキA (50枚)] (Link)                         [学習] [編集]   |
  |  - [デッキB (30枚)] (Link)                         [学習] [編集]   |
  |                                                                      |
  +----------------------------------------------------------------------+
  ```

### 2.2. 【画面2】復習画面
- **役割:** ユーザーが学習に集中するための distraction-free な環境を提供する。
- **状態A：表面（問題）表示**
  ```
  +----------------------------------------------------------------------+
  | `進行状況`: 1 / 25 (ProgressBar)                         [終了] (Link) |
  +----------------------------------------------------------------------+
  |                                                                      |
  |  <カード表面> (Card Component)                                       |
  |                                                                      |
  |    `表面のテキスト（Markdownレンダリング）`                            |
  |    `表面の画像（存在する場合）`                                        |
  |                                                                      |
  |  [ 答えを見る ] (Button)                                           |
  |                                                                      |
  +----------------------------------------------------------------------+
  ```
- **状態B：裏面（答え）表示 ＆ 自己評価**
  ```
  +----------------------------------------------------------------------+
  | `進行状況`: 1 / 25 (ProgressBar)                         [終了] (Link) |
  +----------------------------------------------------------------------+
  |                                                                      |
  |  <カード裏面> (Card Component)                                       |
  |                                                                      |
  |    `裏面のテキスト（Markdownレンダリング）`                            |
  |    `裏面の画像（存在する場合）`                                        |
  |                                                                      |
  |  ---                                                                 |
  |  `理解度を選択してください`                                            |
  |  [ 難しい ] (Button) [ 普通 ] (Button) [ 簡単 ] (Button)            |
  +----------------------------------------------------------------------+
  ```

### 2.3. 【画面3】カード登録 / 編集画面
- **役割:** テキストや画像を使い、直感的に知識カードを作成・編集する。
- **レイアウト:** モーダルダイアログまたは専用ページとして表示。
  ```
  +----------------------------------------------------------------------+
  | `カードの登録`                                         [保存] [閉じる] |
  +----------------------------------------------------------------------+
  |                                                                      |
  |  `デッキ選択`: [ プルダウンメニュー ]                                  |
  |                                                                      |
  |  --- 表面 --------------------------------------------------------- |
  |                                                                      |
  |  [ Markdownエディタ・ツールバー ]                                    |
  |  [ `テキストエリア` ]                                                |
  |  [ `画像アップロードボタン` / `プレビュー` ]                           |
  |                                                                      |
  |  --- 裏面 --------------------------------------------------------- |
  |                                                                      |
  |  [ Markdownエディタ・ツールバー ]                                    |
  |  [ `テキストエリア` ]                                                |
  |  [ `画像アップロードボタン` / `プレビュー` ]                           |
  |                                                                      |
  +----------------------------------------------------------------------+
  ```
